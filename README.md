# Cardiovascular Disease Prediction App

This is a machine learning web application that predicts the risk of cardiovascular disease based on patient health details.

## Project Overview

The project uses a trained deep learning model to classify whether a person has a high or low risk of cardiovascular disease.

The application is built using:

- Python
- Flask
- TensorFlow / Keras
- Scikit-learn
- HTML
- CSS

## Features

- User-friendly web interface
- Takes patient health inputs
- Predicts cardiovascular disease risk
- Uses a trained neural network model
- Includes colorful and responsive frontend design

## Inputs Used

- Age
- Gender
- Height
- Weight
- Systolic Blood Pressure
- Diastolic Blood Pressure
- Cholesterol
- Glucose
- Smoking
- Alcohol Intake
- Physical Activity
